  • Missing smoking pack-years (1,816 of 3,985 patients, ~46%): set to 0. This is a documented assumption, not a confirmed zero for every case — treat pack-year averages as a lower bound.
  • Missing positive lymph node counts (~1,000 patients): set to 0 for the same reason; the "Resource Utilization" averages should be read as a lower bound too.
  • Cancer stage: 498 patients have no recorded pathologic stage. They're counted in every KPI and chart except the Stage breakdown, where they appear as a separate grey "Unknown" bar rather than being silently dropped.
  • City of procurement is not currently shown in any chart, so known typos/casing inconsistencies in that raw column (e.g. "Piitsburgh", "LVIV") were not a priority to clean for this version.
