# **Data Visualization Dashboard**

An interactive dashboard developed using HTML, CSS, JavaScript, D3.js, and Plotly.js.

## **Project Structure**

Keep the project files in the same folder:

```text
project-folder/
├── index.html
├── style.css
├── script.js
├── README.md
├── assets/
└── Esophageal_Dataset_Cleaned.csv
```

## **Running the Project in VS Code**

1. **Open the project**

Open the project folder in Visual Studio Code:

2. **Install Live Preview**

In VS Code:

- Open the Extensions panel.
- Search for `Live Preview`.
- Install the extension published by Microsoft.

3. **Start the dashboard**

- Open index.html.
- Right-click inside the file.
- Select Show Preview.

Alternatively, click the **Show Preview** button in the top-right corner of VS Code.

**Important**

Do not open `index.html` by double-clicking it directly. Some browsers may block the dashboard from loading CSV or JSON files because of local file security restrictions.

Always run the project using Live Preview or another local development server.
